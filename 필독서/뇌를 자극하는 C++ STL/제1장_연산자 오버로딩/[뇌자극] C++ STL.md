# 1장. 연산자 오버로딩

1. 연산자 오버로딩이란
    - 오버로딩을 통해 사용자 정의 타입(클래스 타입) 끼리 연산이 가능하게 만든다
2. 연산자 오버로딩 정의 및 사용하기 
    - 클래스 내부 함수 구현을 통해 사용자 정의 연산을 가능하게 한다
    ex) A.operator+(B) → A + B
3. 단항 연산자 오버로딩
    - 전위 연산자
    ex) A.operator++() → ++A
    - 후위 연산자
    ex) B.operator++(0) → B++
